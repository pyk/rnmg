Random Number Memory Game

Petunjuk Permainan:

  1. Klik tombol Mulai Permainan untuk memulai permainan
  2. Kombinasi angka akan muncul dan hilang dalam beberapa detik
  3. Ingat kombinasi angkanya, lalu masukkan kombinasi angka sesuai ingatanmu ke input dan tekan enter
  4. Ketika benar, level permainan akan naik
  5. Semakin tinggi level, kombinasi angka akan semakin banyak
  6. Jika kamu salah mengingat angkanya sebanyak 3x maka kamu kalah

Selamat bermain! Kembali ke awal
